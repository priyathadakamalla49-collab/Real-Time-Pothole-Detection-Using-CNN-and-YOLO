# Real-Time Pothole Detection using CNN and YOLOv8

## Project Overview

Developed a computer vision system that first classifies road images using CNN and then detects potholes using YOLOv8.

## Tools Used

- Python
- TensorFlow
- Keras
- OpenCV
- YOLOv8
- NumPy

## Workflow

1. Data Collection
2. Data Preprocessing
3. CNN Classification
4. YOLOv8 Detection
5. Evaluation

## Results

- Classified road images as plain or pothole affected.
- Detected pothole locations using bounding boxes.
